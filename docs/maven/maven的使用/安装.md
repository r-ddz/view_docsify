
## 1 下载

> 搜索官网下载解压即可，无需安装（前置条件：JDK环境）。

官网地址：https://maven.apache.org/

## 2 配置环境变量

### 2.1 MAVEN_HOME 配置

MAVEN_HOME 配置 maven 的目录，例如：

> D:\home\app\Maven\apache-maven-3.6.0

### 2.2 MAVEN_HOME 配置

Path 增加配置 maven 的bin，例如：

> %MAVEN_HOME%\bin

### 2.3 验证maven环境配置是否成功

> 打开 cmd 终端输入 mvn -version 查看版本号。
