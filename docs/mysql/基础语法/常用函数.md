## 时间日期相关

```sql
-- 获取日期：当天
SELECT CURDATE() FROM DUAL;
-- 获取日期：明天
SELECT DATE_SUB(CURDATE(),INTERVAL -1 DAY);
-- 获取时间：今天1点30分
SELECT DATE_ADD(DATE_SUB(CURDATE(),INTERVAL 0 DAY),INTERVAL 90 MINUTE);
-- 获取时间：昨天1点
SELECT DATE_ADD(DATE_SUB(CURDATE(),INTERVAL 1 DAY),INTERVAL 1 HOUR);
```

## 匹配函数 CASE WHEN 语法

```sql
SELECT 
  (CASE 
    WHEN 1='1' AND 1=1 THEN '满足条件1' 
    WHEN 1 = '3' THEN '满足条件2' 
    ELSE '都不满足' 
  END) AS 'CASE语法测试'
FROM DUAL;
```
